# Feature Backlog

Structured feature proposals for Strait of Hormuz Defense.

## Status Labels

| Status | Meaning |
|--------|---------|
| `draft` | Idea captured, needs design |
| `ready` | Designed and scoped, ready for implementation |
| `in-progress` | Currently being built |
| `shipped` | Live in the game |
| `cut` | Decided against — kept for context |

## Priority Labels

| Priority | Meaning |
|----------|---------|
| `P0` | Core gameplay — game doesn't work without it |
| `P1` | High impact — significantly improves experience |
| `P2` | Nice to have — improves polish or depth |
| `P3` | Future — good idea, not needed now |

## File Naming

```
NNNN-short-slug.md
```

- `NNNN` = sequential number (0001, 0002, ...)
- `short-slug` = kebab-case summary

## Template

Every feature file follows the template in `_template.md`.
